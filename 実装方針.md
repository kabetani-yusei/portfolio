# ポートフォリオサイト ゼロからの実装方針

## 設計思想

企業のコーポレートサイトや採用LPのような**クリーンで洗練されたデザイン**を目指す。
シングルページ構成で、スクロールするだけで全情報が読める構造にする。

## 技術スタック

- Next.js 16 (App Router) / React 19 / TypeScript — 既存設定を流用
- Tailwind CSS 4 — 既存設定を流用
- **framer-motion** — スクロール連動のフェードインアニメーション（追加）
- **lucide-react** — 軽量アイコンセット（追加）
- フォント: Inter（英語）+ Noto Sans JP（日本語）

## ページ構成

**シングルページ（`/`のみ）** にすべての情報を集約する。
既存の `/achievements`, `/experience`, `/timeline` ページは削除。

セクション構成（上から順に）:
1. **Hero** — 名前・肩書き・写真・一言紹介・SNSリンク
2. **About** — 研究分野・所属
3. **Skills** — 技術スタック + 資格
4. **Achievements** — 実績（カテゴリフィルタ付き）
5. **Experience** — その他の経験・活動
6. **Timeline** — 経歴の時系列表示
7. **Footer** — 各種アカウントリンク + コピーライト

## デザイン詳細

### カラーパレット

| 用途 | 色 | Tailwind |
|------|------|----------|
| 背景（メイン） | `#ffffff` | `white` |
| 背景（セクション交互） | `#f8fafc` | `slate-50` |
| テキスト（見出し） | `#0f172a` | `slate-900` |
| テキスト（本文） | `#475569` | `slate-600` |
| テキスト（補助） | `#94a3b8` | `slate-400` |
| アクセント | `#2563eb` | `blue-600` |
| アクセント背景 | `#eff6ff` | `blue-50` |
| ボーダー | `#e2e8f0` | `slate-200` |

### タイポグラフィ

- 見出し（h1）: Inter 48px bold
- 見出し（h2）: Inter 32px semibold（セクションタイトル）
- 本文: Noto Sans JP 16px regular
- キャプション: 12px uppercase tracking-wide（セクションラベル）

### レイアウト原則

- 最大幅: `max-w-6xl`（1152px）
- セクション間: `py-20` 以上の余白
- カード: 白背景 + `border-slate-200` + `shadow-sm`、ホバーで `shadow-md`
- 背景が白のセクションと `slate-50` のセクションを交互に配置して視覚的に区切る

### ナビゲーション

- 画面上部に固定（sticky）
- スクロールするとセクション名がハイライトされる
- 白背景 + 下線ボーダー + `backdrop-blur`
- クリックでスムーススクロール

### アニメーション

- framer-motion の `whileInView` で各セクション・カードがスクロール時にフェードイン
- ホバー時のカード浮き上がり（`translateY(-2px)` + shadow変化）
- ナビリンクのアンダーライントランジション
- 派手な動きは一切不要。上品で控えめに

│   ├── Achievements.tsx     # 実績セクション（フィルタ付き）
│   ├── Experience.tsx       # 経験セクション
│   ├── Timeline.tsx         # タイムラインセクション
│   ├── Footer.tsx           # フッター

## 各セクションの詳細

### Hero

- 2カラム: 左にテキスト、右にプロフィール写真（`/user.jpg`）
- 名前を大きく、英語名をサブテキストで
- 肩書き: 「名古屋大学大学院 情報学研究科 数理情報学専攻 修士1年」
- 一言: 「組合せ最適化を研究しています。競技プログラミング、機械学習コンペ、ハッカソンに参加しています。」
- SNSアイコンリンク（GitHub, X, AtCoder 等）を横並びで
- 写真は角丸 + 薄いシャドウ

### About

- 研究分野カード + 所属一覧
- シンプルな2〜3カラムのカードレイアウト

### Skills

- 2カラム: 左にスキル（タグ形式）、右に資格（リスト形式）
- スキルタグ: `bg-blue-50 text-blue-700 border border-blue-200` のピル型
- 資格: アイコン + テキストのリスト

### Achievements

- セクション上部にカテゴリフィルタボタン（All / Hackathon / Data Science / AI Competition / AtCoder / Academic）
- カード形式で一覧。カテゴリタグ + 日付 + タイトル
- フィルタは `useState` で管理（`"use client"` コンポーネントとして切り出し）

### Experience

- カード形式のグリッド（2列）
- SIGNATE Grandmaster、AtCoder 水/青 などの特筆事項は視覚的に目立たせる

### Timeline

- 左に縦線 + ドット、右にカード
- 上から新しい順に並べる
- 学歴・職歴・修了プログラムをまとめて表示

### Footer

- 各種アカウントリンクをアイコン + テキストで横並び
- `© 2026 Yusei Kabetani` + 技術スタック表記

## 実装順序

1. 追加パッケージインストール
2. 既存の `src/` 配下を全削除
3. `src/data/profile.ts` — データ定義
4. `src/app/globals.css` — グローバルスタイル
5. `src/app/layout.tsx` — ルートレイアウト
6. `src/components/FadeIn.tsx` — アニメーションユーティリティ
7. `src/components/SectionHeading.tsx`, `Tag.tsx` — 汎用コンポーネント
8. `src/components/Navbar.tsx` — ナビゲーション
9. `src/components/Hero.tsx` — ヒーローセクション
10. `src/components/About.tsx` — Aboutセクション
11. `src/components/Skills.tsx` — スキルセクション
12. `src/components/Achievements.tsx` — 実績セクション
13. `src/components/Experience.tsx` — 経験セクション
14. `src/components/Timeline.tsx` — タイムラインセクション
15. `src/components/Footer.tsx` — フッター
16. `src/app/page.tsx` — メインページ（全セクション統合）
17. ビルド確認・微調整
